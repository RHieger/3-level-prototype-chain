# Demonstration of a 3-Level Prototype Chain

This demonstration is based on several lectures in the course, **_JavaScript Advanced&mdash;Prototypes,
Inheritance &amp; Objects_** with instructor Clyde Matthew, and hosted on Udemy.com.

In an attempt to go into depth with what might be a more real-world example based on that shown
in the lectures, I have created an executable program that describes step by step the process
of building the prototype chain.

The code is by no means optimized, but by virtue of a dependency called chalk, I was able to
produced timed output of color-coded messages that track the entire development of the chain.

This program is executable by issuing the command:

```javascript
npm start
```

## NOTE:

The color-coded output, as mentioned above, is implemented using a package called chalk,
which is avalable through NPM at this address:

[https://www.npmjs.com/package/chalk](https://www.npmjs.com/package/chalk)

Using this package, instances of the chalk object were created, each allowing for a color coding
of either explanatory messages or reproduction of color-coded code.
